# List Support Test

## Unordered Lists

Using hyphens:
- First item
- Second item
- Third item

Using asterisks:
* First item
* Second item
* Third item

Using plus signs:
+ First item
+ Second item
+ Third item

## Ordered Lists

1. First numbered item
2. Second numbered item
3. Third numbered item

## Mixed Lists

Here's an unordered list:
- Item one
- Item two
- Item three

And here's an ordered list:
1. Step one
2. Step two
3. Step three

## Lists with formatting

- **Bold item**
- *Italic item*
- ~~Strikethrough item~~
- `Code item`
- [Link item](https://example.com)

1. **Bold numbered item**
2. *Italic numbered item*
3. ~~Strikethrough numbered item~~
